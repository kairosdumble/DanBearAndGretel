# 프로그램 실행방법

1. backend
backend부터 실행(서버 구동)

- (윈도우 기준) cmd창에서 cd backend를 입력
- npm run dev 입력

2. frontend
애뮬레이터나 실제기기 연결 시
- cd frontend를 입력
- npm run dev 입력

웹으로 보기
- flutter run 입력

api 키는 frontend/lib/tmap_view.dart에서 7줄   _tMapApiKey변수로 들어가 있습니다.
